# BBSforum
  基于Springboot+Spring+Mybatis(SSM)框架的一个简易BBS论坛，使用Websocket协议与前端进行实时交互。
  
## 主要技术

- 后端技术：spring、Springboot、Mybatis

- 缓存（Nosql）:Redis

- 数据库：MySQL

- 服务器： Nginx（web服务器）和tomcat

- 版本控制：git
 
## 开发工具和环境
- IntelliJ IDEA 2017.3.2

- Maven 4.0.0

- Apache Tomcat/8.5.16（Maven Tomcat Plugin）

- JDK 1.8

- Mysql 5.6

- Nginx 1.5.1

- Win10 操作系统
